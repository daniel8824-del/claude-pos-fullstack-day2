# 상권 분석 스킬

2. 매장별 매출 (막대 차트)
3. 인기 상품 Top 5 (도넛 차트)

## 색상 규칙

- 강조 = 빨강 #c92a2a

1핵심 개념 | HOOK

개념정의
HOOK~/.claude/hooks/ 이벤트 트리거 | PreToolUse / PostToolUse / SessionStart 등. 코드 저장·실행 시 자동 동작.

/security-review슬래시 명령 OWASP Top 10 기준 자동 보안 검수 | 비밀번호 노출 · SQL 인젝션 · 민감 정보 등 점검.

2실습 | 자주 쓰는 훅 5종

훅역할
carefulrm -rf 등 위험 명령 자동 차단
compact-guard컨텍스트 50% 도달 시 자동 압축
smart-commitgit commit 직전 메시지 자동 점검
replay-learnings세션 시작 시 과거 교훈 자동 회상
learn-rule"기억해" 키워드 감지 시 자동 저장
